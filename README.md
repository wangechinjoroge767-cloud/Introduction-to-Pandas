# Introduction to pandas  
- Pandas is a Python library used for data analysis, data cleaning and data manipulation.  
- It helps us work with structured data especially data in table form, similar to Excel, SQL tables or CSV files.  

_Analogy_ Pandas is a Python Excel.   

In pandas you can:  
- Open data
- Filter through the data  
- Create new columns  
- Clean missing values  
- Sort data  
- Group data  
- Do calculations on the data.
